# Documentacio obsoleta o historica

Aquesta carpeta conserva documentacio antiga, generada o de fase/milestone que ja no es considera font activa del projecte.

No s'ha eliminat per mantenir context historic i traçabilitat, pero no s'ha d'usar com a estat actual sense verificar contra:

- `AGENTS.md`
- `MEMORIA_PROJECTE.md`
- `README.md`
- `docs/database.schema.md`
- les migracions de `supabase/migrations/`
- la BD real, quan calgui
